# Sales Data Cleaning and Dashboard (Excel)

## Project Overview
This project demonstrates how raw sales data was cleaned, standardized, and analyzed using Microsoft Excel.

The dataset contained:
- Missing values
- Inconsistent date formats
- Text in numeric columns
- Inconsistent country and gender values

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Functions (IF, TRIM, PROPER, IFERROR, DATEVALUE)
- Charts and Dashboard Design

## Data Cleaning Steps
- Standardized customer names
- Cleaned sales amount and removed symbols
- Unified date formats
- Replaced missing values
- Standardized country and gender values

## Dashboard Insights
- Total sales performance
- Sales by country
- Sales by gender
- Sales trends over time
- Customer age distribution
